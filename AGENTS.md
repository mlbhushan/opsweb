# Writing Style Rules

- **Never use em dashes** (the — character) in any written text, copy, descriptions, or code comments. Use commas, colons, or restructure the sentence instead.
- **Brand Fonts Only**: Always stick to the brand fonts explicitly established in the project (e.g., TASA Orbiter, Inter, or specific sans-serif). Never use system fonts like `font-serif` which default to Georgia/Times New Roman and break the brand aesthetic.
